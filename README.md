
# 🎥 Videotron Analytics

Central documentation for the Videotron Analytics system, including architecture, API, data model, AI pipeline, and infrastructure design.

---

## 📌 Overview

Videotron Analytics is a system designed to analyze audience interaction and traffic around digital signage (videotron).  
The system leverages computer vision and data processing to provide insights such as:

- Audience detection
- Attention analysis
- Traffic estimation
- Behavioral insights

---

## 🎯 Objectives

- Measure audience engagement with videotron displays
- Provide data-driven insights for advertisers
- Support decision-making using real-time and batch analytics

## 🧠 Architecture

High-level system flow:

1. Camera captures videotron area
2. Data is processed (batch or real-time)
3. AI model analyzes audience & behavior
4. Results are stored and visualized in dashboard
